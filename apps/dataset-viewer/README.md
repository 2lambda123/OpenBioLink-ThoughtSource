## Setup

1. Clone repository
2. Install the dataloader `pip install -e ./libs/dataloader`, see [here](../../libs/dataloader/README.md)
3. Install streamlit `pip install streamlit`
4. Run `streamlit run ./apps/dataset-viewer/app.py`
